Automate end-of-day workflow for Obsidian daily notes. Load `daily-wrapup` skill for full workflow instructions.

**Process**:
1. Read today's daily note (daily/YYYY-MM-DD.md)
2. Interview user about tomorrow's plans for Next Episode section
3. Analyze completed Tasks for learning opportunities (decisions, patterns, learnings)
4. Extract ideas for new/existing notes
5. Update daily note with Next Episode and Ideas for Notes sections
6. Create concept notes in work/concepts/ using standard structure
7. Link created notes back to daily note in Ideas for Notes section

**References**:
- See skill at `~/.config/opencode/skill/daily-wrapup/`
- Interview questions: `references/interview-questions.md`
- Idea extraction patterns: `references/idea-extraction.md`
