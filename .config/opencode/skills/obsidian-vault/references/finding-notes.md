# Finding Notes

## Vault Structure Overview

```
notes/
├── topics/          # 9 topic files + Index.md
├── work/
│   ├── concepts/    # 18 concept files
│   ├── tickets/     # 33 ticket files
│   ├── retros/      # Retrospective notes
│   └── interviews/  # Interview assessments
├── daily/           # Daily journal entries
├── templates/       # Reusable templates
└── archives/        # Archived notes
```

## Search Strategies by Domain

### Data Engineering
Start at `topics/` → look for DE-related topics → follow links to concepts

### GIS
Check `topics/GIS.md` → explore linked concepts in work/concepts/

### AI/GenAI
Navigate through `topics/` → find AI-related topics → discover concepts

### Work Tickets
Go directly to `work/tickets/` or use `RD-` prefix search

## Using Index.md

The `topics/Index.md` serves as the vault's navigation hub. It contains:
- Overview of all topic areas
- Links to major documentation sections
- Entry points for each domain

## Link Discovery

With 157 wikilinks in the vault:
1. Start from a relevant topic or concept
2. Follow wikilinks to discover related content
3. Use backlinks panel to find what links to current note
4. Search for filename patterns when unsure of location
